The following functions are available globally.

  • saveContextAndWait(_:)

    Attempts to commit unsaved changes to registered objects to the specified context’s parent store. This method is performed synchronously in a block on the context’s queue. If the context returns false from hasChanges, this function returns immediately.

    Declaration

    Swift

    public func saveContextAndWait(context: NSManagedObjectContext) -> ContextSaveResult

    Parameters

    context

    The managed object context to save.

    Return Value

    A ContextSaveResult instance indicating the result from saving the context.

  • saveContext(_:_:)

    Attempts to commit unsaved changes to registered objects to the specified context’s parent store. This method is performed asynchronously in a block on the context’s queue. If the context returns false from hasChanges, this function returns immediately.

    Declaration

    Swift

    public func saveContext(context: NSManagedObjectContext, completion: (ContextSaveResult) -> Void)

    Parameters

    context

    The managed object context to save.
    completion

    The closure to be executed when the save operation completes.

  • deleteObjects(_:inContext:)

    Deletes the objects from the specified context. When changes are committed, the objects will be removed from their persistent store. You must save the context after calling this function to remove objects from the store.
