FetchRequest

An instance of FetchRequest <T: NSManagedObject> describes search criteria used to retrieve data from a persistent store. This is a subclass of NSFetchRequest that adds a type parameter specifying the type of managed objects for the fetch request. The type parameter acts as a phantom type.
