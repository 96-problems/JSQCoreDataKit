FetchResult

A FetchResult represents the result of executing a fetch request. It has one type parameter that specifies the type of managed objects that were fetched.

  • success

    Specifies whether or not the fetch succeeded.

    Declaration

    Swift

    public let success: Bool
  • objects

    An array of objects that meet the criteria specified by the fetch request. If the fetch is unsuccessful, this array will be empty.

    Declaration

    Swift

    public let objects: [T]
  • error

    If unsuccessful, specifies an error that describes the problem executing the fetch. Otherwise, this value is nil.

    Declaration

    Swift

    public let error: NSError?
